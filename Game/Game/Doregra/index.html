クイズ　ドレミファグランプリ（コナミ）
Last Update 1998.05.05

    TVのドレミファドンを参考にして作られたイントロ当てクイズです。この後、本作の続編として２，３が作られていますが、シンプルさ・面白さにおいては本作が一番だと思います。

    出題パターン
    種類内容
    イントロクイズ 出題時に選択肢が表示される。
    イントロが流れる間にその曲のタイトルを当てる。
    残り時間が得点となる。最高１００点。
    ？イントロクイズ イントロが流れる間にその曲のタイトルを当てる。
    残り時間が得点となる。最高１００点。
    イントロクイズと違い、出題時に選択肢が表示されない。
    ウルトライントロクイズ 出題時に選択肢が表示される。
    イントロがまず１．５秒、次に２．５秒間流れる。
    1回目で答えると１００点、2回目で答えると５０点。
    超ウルトライントロクイズ 出題時に選択肢が表示される。
    イントロがまず０．８秒、次に１．５秒間流れる。
    1回目で答えると200点、2回目で答えると1５０点。
    面クリアタイプのクイズゲームで、クイズ内容はイントロを聞いて曲名を答えるだけのシンプルなクイズです。が、ＡＬＬクリア時のみしかネーム入れできない、ネーム入れにはかなり厳しいゲームでもあります。全５ステージ。問題数は不明。

    ゲームシステム

    • 各面の終了時、合格点以上の点数をその面で取れれば面クリアとなります。
    • 全５ステージをクリアする事でＡＬＬクリアとなります。
    • 各ステージの始めにジャンル選択をします。
    • 出題パターンは４種類。
      回答ボタンを押して回答権を得てから改めて答えのボタンを押す形式です。時間切れのペナルティはなし。回答権を得てからの時間切れおよび、回答ミスのペナルティは−５０点。
    ジャンル名説明
    最近の流行曲が得意９０年代の曲が中心
    ちょっと昔の曲が得意８０年代の曲が中心
    懐かしい曲が得意７０年代の曲が中心
    最新ヒットから懐メロまで全ての年代の曲から出題

    ジャンル選択について

    • ジャンル選択は各面の初めにします。
    • ４つのジャンルの中から１つを選べます。
    • 直前の面に選択したジャンルを選択することはできません。

    各ステージについて

    このゲームでは、各ステージのノルマおよび、出題形式が固定で決まっています。
