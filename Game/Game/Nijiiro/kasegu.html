得点を稼ぐには（基礎編）

得点を稼ぐにはどういう方法が考えられるか？システム的に見て３つのポイントがあります。

★クイズにできるだけ早く回答する

より高いタイムボーナスを取る為に、早く回答します。最高で１問１００００点のタイムボーナスを取ることが出来ます。（まぁ、普通にやってると不可能ですが。）
ただし、このクイズの場合、以下の各種要因により、タイムボーナス１００００点を取る事が困難であるといえます。
  1. タイムボーナスが取れる１コマ目が０．２秒という、極めて短時間の回答時間しかない。
  2. 回答欄が２×２段と成っていて、回答枝の長さで回答する等の方法が取りにくい。
  3. 問題出題が瞬間的に行われ、タイムボーナスのカウントダウンのタイムラグが全くない。

★マップをなるべく少しずつ進む

マップ画面を少しずつ進む事で、回答する問題数を増やし得点を稼ぎます。 こうすると、キャラクター毎の好感度も増加するので、特殊イベントも発生して回答する問題数が増え、更に得点を稼ぐことができます。 ルーレットを目押しして行く事で対処します。
しかし、このゲーム、そう簡単には目押しをさせてくれません。それはファンブルと呼ばれる人為的な目押しずれです。目押しをして、１回目と２回目は１００％目押しを成功させられるのですが、３回目では約３割方、または４回目では絶対に失敗してしまいます。（３回目で失敗する確率はクイズの正答率に寄っている様で、連続正解していると、ほとんど３回目で失敗する様です。）
ファンブルというのは単純考えると目押しした目に対して規定の回転が加わって、目が決定されるというだけなので、「いつ」「どれだけ」のファンブルが起こるかが分かればよいのですが、そうでない内はどうしようもないですよね。

★２倍アイテムを早い時点でＧＥＴし、なくさない

２倍アイテムを用いる事で、その時に取るタイムボーナスを２倍にする事が出来ます。うまくすれば、最高で１問２００００点を得る事が出来ます。ゲーム開始時点から持っている事が一番望ましいです。不幸にして持っていない場合は、途中ではやめにＧＥＴし、その後は間違えず、なるべくすばやく答える事で対処します。

得点を稼ぐには（実践編）

基礎編ではどうすると稼げるか？という話でしたが、ここではどうやって実現するか？という話です。

★電源パターンの基礎

普段であれば、基礎編で書いたような事を人間業でずっとこなすのはとても不可能だと思います。多分、１５０万やそこら行けばいいところでしょう。でもこのゲーム、１５０万どころか、１０００万近くまで点数が行っています。 さてどうやって？と思うでしょう。この答えが電源パターンの存在です。
電源パターンとは本来、電源を入れてから同じタイミングで同じ操作を行う事で、毎回ゲーム中の何かが同じ結果となるやり方の総称です。
では、このゲームの電源パターンで同じになるものは何か？ということになりますが、それは以下の３点に集約できます。
  1. 出現するアイテム
  2. クイズの正解位置
  3. ルーレットの回転
つまり、
  1. 出現するアイテムを調整する事で２倍アイテムを確実に出し、
  2. クイズの正解位置を調べる事でボーナス１万点（２倍使用時は２万点）を確実に得て、
  3. ルート選択を調べる事でなるべく長いルートを進み、止まるマスを増やす。
  4. ルーレットの回転を調整する事でマップをなるべく少しずつ進み、止まるマスを増やす。
という、３段論法？により、無事ハイスコアをＧＥＴと言う訳です。

★出現するアイテムを調整する

電源パターンで出現するアイテムを調整するには色々な方法が考えられます。
自分の場合は、
  1. 最初のコインは最速のタイミングで投入し、ゲーム途中で問題をミスして、途中のアイテムを変える
  2. 最初のコインを投入するタイミングを調整し、出現時のアイテムを変える
という、２種類の方法を使用したパターンを作成していました。

★クイズの正解位置を調べる

これは特に考える事はありません。調べるだけです。記録する時、間違えずに記録するだけです。あとは毎回同じ様に実行すれば、自動的に１問１万点（２倍使用時は２万点）のタイムボーナスが入ります。

★ルート選択を調べる

このゲームは、カレンダーを模したＭＡＰになっているで、どこを通っても余り変わりませんが、同時に色々なキャラの好感度をＵＰさせられるルートを選ぶのがいいでしょう。自分の場合は江崎ルートと、めぐみルートを選択していました。

★ルーレットの回転を調整する

これが今までの中で一番技術的に問題となる項目でしょう。まず、考えられる選択としては、
  1. 押さないで待つ
  2. 押しっぱなし
  3. 押さないで待つ＆押しっぱなしの組み合わせ
  4. ファンブルを見越して「１」を目押しする
です。
「押さないで待つ」というのは文字通り、勝手に止まるまで待つというものです。放っておく事で、毎回のルーレットの出目を安定させます。技術的には簡単なので、パターンが完成すれば、ミスが少ない方法です。ただし、悪い目が出てもルート調整でしか調整不能なので、そこが弱点です。
「押しっぱなし」というのはボタンに連射を掛け、ルーレットをすぐに止めようとするものです。通常、同じタイミングで止める事は無理ですが、シンクロ連射等を使う事で、理論上安定してタイミングで止める事ができ、毎回のルーレットの出目を安定させることができます。この方法も技術的には簡単なので、パターンが完成すれば、ミスが少ない方法です。ただし、悪い目が出てもルート調整でしか調整不能なので、そこが弱点です。
「押さないで待つ＆押しっぱなしの組み合わせ」というのは、上記の方法を組み合わせ、より多くのマスに止まる組み合わせを考えるというものです。基本的にはどちらか片方の、より少ない目で進む方を選択して行きます。ただ、チョットでかい目がでても、先々で少ない目が連続した場合というのも考えられるので、その組み合わせは無限大にあると考えられます。技術的には簡単なので、パターンが完成すれば、ミスが少ない方法です。
「ファンブルを見越して目押しする」というのは、このゲームで考えられる、最良の方法です。ファンブル自体、発生する場面により、決まっている為、それを見越し、わざとずらした所を狙って目押しをして「１」を出します。この方法は理想的ですが、技術的にはかなり難しいので、パターンとして安定させるには工夫が必要になると思います。
ちなみに、自分の場合は機械的に目押しをする方法を考えて、安定させていました。目押しのうまい人ならば、普通に目押しすればいいかもしれないです。
