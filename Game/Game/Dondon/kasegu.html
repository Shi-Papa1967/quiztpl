高得点を狙うための攻略

高得点を狙うにはどうするか。色々考察していきます。

★問題をできるだけ速く答える

まず考えられるのは、問題をできるだけ速く答えることです。それもいいでしょう。これで試算すると以下の様になります。
各面のノルマの合計（５枚＋１３枚ｘ８面）ｘ５０００点＝５４５、０００点

★パネルめくりボーナスを狙う

第２にパネルめくりボーナスを狙うことを考えてみます。いっぺんにめくる相手のパネルの枚数が多いと高得点となるパネルめくりボーナス（最大５１１，０００点のパネルめくりボーナスが入る。）は、このゲームで高得点を取る上で非常に重要な位置を占めています。 「坊主パネル」が出た時にうまく間違えて自分のパネルを相手のパネルにしてやり、その後、自分がもう一度パネルで挟んで自分のパネルにしてやれば、パネルめくりボーナスはオレのものというわけです。ただ、いつ「坊主がパネルが出る」か分からないため、非常にバクチの要素が大きいやり方でもあります。
「坊主パネル」を使ったやり方としては、次の様なものが考えられます。
  1. 普段は早押しで得点を稼ぐ。
  2. 「坊主パネル」が偶然出た時に、ワザと間違え、コンピュータのパネルを増やす。
  3. うまく挟んで一度に多くのパネルを取り返す様にする。
  4. 挟めない時のパネルの選択は、コンピュータのパネルを取らないで、かつ次にコンピュータのパネルを取れる位置のパネルを取る。
このやり方で行くと、タマに引く「坊主パネル」により、若干点が伸びる要素があるため、ただ単純に答えていくよりはずっと高得点を狙える為、よりすぐれたやり方であるといえます。

●…自分のパネル、番号…相手のパネル、◎…最後に自分が置くパネル
「坊主パネル」を利用して稼ぐとして、さてどのパターンでめくるのが一番パネルをめくれるのでしょう？パネルめくりボーナスはよりたくさんの相手のパネルを１回でめくればめくるほど、高得点が得られます。 このゲームのフィールドは５ｘ５なので、図のように自分と敵のパネルを配置できれば、最大９枚、合計５１１０００点がはいりますね。 １面最大１回しかやれませんが、これが最上のパネルのめくり方となります。 ただ、これを実行するには、
  1. 相手のパネルにするべき所（番号の所）を取る。
  2. 「坊主パネル」がでて、間違える。
  3. 自分のパネルにすべき所（●の所）を取る。
  4. ◎の所を取る。
と、４段階の過程を踏まないといけないですが、「坊主パネル」が出る位置が決まってないので、事実上不可能に近いです。ただ、９枚とは行かないまでも、４～５枚めくりとかはできない事はないので、（実際に７枚めくりくらいまではやったことがあります。）このやり方が普通にやる分には最上のやり方であるといえます。

電源パターンを使った高得点を狙う攻略

電源パターン。電源を切ってからゲームを始めると、でてくる問題等などが同じになってしまうというやり方です。このゲームの場合、パネルの配置・問題とその回答位置など、このゲームのクイズ部分に関しては、ほとんど同じとなってしまいます。（ボーナスゲームはパターンにならなかったですが。）
これを使って上記のやり方のベスト追求するパターンを作ればと、考えた訳です。

★このゲームにおける電源パターン

このゲームでは電源を切ってからゲームを始めると、 パネルの選択時間を一定にするという条件を満たしている限り、下記の項目が常に一定になります。
電源パターンで一定となる項目
特殊パネルの配置（面出現時に決定）、ジャンルセレクトパネルの位置（面出現時に決定）、出題される問題、解答枝の位置、デフォルトで動くカーソル位置、ＣＰＵの動かすカーソルの位置

条件には「自分でカーソルを動かす事」や、「問題の解答時間」がありませんので、自分で自由にカーソルを動かしていいという事になります。また、特殊パネルのパターンは面出現時に決定されるので、予め「坊主」位置を調べておき、目的の９枚をめくった後に「坊主」を引いて間違えるという手順が可能になります。
パターン作成上の問題点は以下のようなものがあげられます。
  1. 目的の９枚をめくった後にクリアせずに「坊主」が選択できる位置に有るのか。
    このゲームのノルマは１３枚、パターン作成の都合上、９枚はパネルをめくらないといけません。このため、変な位置に「坊主」パネルが有る時にどうするのかという問題です。
  2. パネルが全て敵のものになった後で、必要なパネルをめくらない様にして、●のパネルを選択できるのか。
    ●のパネルはなるべく多くのパネルをめくる為、離れた位置となっています。しかし、基本的には自分のパネルの周りしか次に取るパネルを選択できないので、気を付けないと●のパネルをめくるべく、自分のパネルを取っているうちに、必要なパネルをめくってしまうということも起こってしまいます。
これを回避する方法として、考え出したのが、ワザと間違えるというものです。というより、パターン作ってる途中に必要なパネルをめくってしまったが為、どうしようもなくって、回避する手段として試したのですが．．．これがドンピシャで、うまくコンピュータが動いてくれたので、無事採用となりました。（多分どうしようもなかったら、パネルの選択時間を調整しただろうと思う。）
